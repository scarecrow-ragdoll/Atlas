# AI Review History

## Source Evidence

PRD §19.

## User Problem

Store and track AI recommendations over time to see what was advised, what was applied, and how decisions evolved.

## Scope

In MVP. Simple manual entry, no API integration.

## Behavior

- User creates review entry
- Pastes AI response text
- Links review to a date range
- Adds notes and planned actions
- Viewable history of past reviews

## Acceptance Criteria

AC-025, AC-090 through AC-092.
